----- passo a passo ---------
Deve-se instalar inicialmente:
- selenium para python
    pip install selenium

- apt-get install python-pip
- apt-get install xvfb xserver-xephyr vnc4server
- pip install pyvirtualdisplay

Deve-se baixar o Driver correspondente ao navegador que será utilizado para os teste, neste caso foi chrome e copiá-lo para /usr/bin
    https://sites.google.com/a/chromium.org/chromedriver/downloads
